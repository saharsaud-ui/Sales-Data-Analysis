# 📊 Sales Performance Analysis

## 🎯 Project Goal
The main objective of this project is to analyze a month's worth of electronics sales data to answer key business questions:
* What was the best-selling product?
* Which city had the highest number of sales?
* What are the revenue trends across different categories?

## 🚀 Technical Skills Demonstrated
* **Data Cleaning:** Handling missing values and filtering out "dirty" data records.
* **Feature Engineering:** Extracting cities from addresses and calculating total revenue.
* **Exploratory Data Analysis (EDA):** Using `groupby` and `sum` functions to aggregate data.
* **Data Visualization:** Creating professional bar charts using `Seaborn` and `Matplotlib`.

## 📦 Libraries Used
* **Pandas:** For data manipulation.
* **Matplotlib & Seaborn:** For creating high-quality statistical graphics.

## 💡 Top Findings
* **Market Leader:** San Francisco is the most profitable city, contributing significantly to the total revenue.
* **Hero Product:** The MacBook Pro 16-inch is the primary revenue driver, outperforming other electronics.
* **Optimization:** Analyzing purchase patterns helps in stock management and targeted marketing.
